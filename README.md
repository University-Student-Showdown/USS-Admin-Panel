# Admin Dashboard for USS Tournaments (Whilst website is in development)

## Requirements
- Java (18+)
- A Google account
- Correct credentials (Explained later)

## Setup
- Admins will be given a corresponding ```token.json``` file which will contain data to access and manipulate the sheet
- The Program can be ran by downloading the latest release, and running the jar file found.
- On first start up, this should prompt you with a google window to gain access to the sheet
